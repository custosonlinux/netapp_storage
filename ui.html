Data Backup / Restore

Export plugin configuration (endpoints, PVE hosts, volume mappings, schedules, provisioned datastores, SMTP config) as a JSON file. Use the exported file to restore the plugin on a new or rebuilt server. Passwords are exported in encrypted form — they will only work if the encryption key is the same on the target server. Snapshot history and SnapMirror relationship cache are not included — both are repopulated automatically.
